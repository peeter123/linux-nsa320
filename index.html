linux-nsa320 by psch2

Linux support for ZyXEL NSA320 2-Bay Power Media Server

Currently there exits a board setup and a hwmon platform device driver (nsa3xx-hwmon).

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/psch2/linux-nsa320

get the source code on GitHub : psch2/linux-nsa320
